# Contributing to LibreDevice OS

এই project-এ তোমার contribution স্বাগত।

LibreDevice OS বিশ্বাস করে — ভালো idea কোনো একজনের একার থেকে আসে না।
তুমি developer, security researcher, hardware engineer, বা শুধু একজন
curious মানুষ হলেও — তোমার চিন্তা এখানে মূল্যবান।

---

## কীভাবে Contribute করবে

### ১. Concept নিয়ে আলোচনা
- **Issues** খুলো — কোনো idea, প্রশ্ন, বা সমস্যা নিয়ে
- বাংলা বা English — যেকোনো ভাষায়

### ২. Document উন্নত করতে
- Fork করো
- নিজের branch-এ কাজ করো
- Pull Request পাঠাও

### ৩. Technical Research
এই বিষয়গুলোতে সাহায্য দরকার:
- Osmocom-based 4G/5G modem open firmware-এর current status
- Microkernel implementation options (seL4, Zircon, etc.)
- UVC camera module compact design
- EMI shielding for tablet-size device
- Traffic correlation attack mitigation

### ৪. Hardware Design
- Modular connector standard proposal
- Piano switch panel PCB design
- Anti-tamper sensor circuit

---

## Contribution-এর নিয়ম

### ✅ করবে:
- Respectful, constructive feedback
- Evidence-based technical argument
- নতুন threat বা solution-এর suggestion
- Document translation (অন্য ভাষায়)

### ❌ করবে না:
- Proprietary solution suggest করবে না যেটা user freedom কমায়
- Privacy-invasive approach suggest করবে না
- অযাচাইকৃত technical claim করবে না

## License Agreement

Contribute করলে তুমি agree করছো যে তোমার contribution:
- **Documentation:** CC BY-SA 4.0 license-এ থাকবে
- **Code:** GNU GPL v3 license-এ থাকবে

এর মানে তোমার contribution চিরকাল মুক্ত থাকবে।

## Core Philosophy মনে রেখো

> "OS তোমার guardian নয় — তোমার সৎ বন্ধু।"
> "প্রযুক্তি মানুষের জন্য, মানুষ প্রযুক্তির জন্য নয়।"

সব contribution এই philosophy-র সাথে সামঞ্জস্যপূর্ণ হতে হবে।
